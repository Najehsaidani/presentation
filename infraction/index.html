2-Logiciel Informatique

Visual Studio Code est un éditeur de code extensible développé par Microsoft pour Windows, Linux et MacOs. Les fonctionnalités incluent la prise en charge du débogage, la mise en évidence de la syntaxe

Visual Studio est un ensemble complet d'outils de développement permettant de générer des applications web ASP.NET , des services web XML, des applications bureautiques et des applications mobiles.

PhpMyAdmin est une application Web de gestion pour les systèmes de gestion de base de données MySQL et MariaDB, réalisée principalement en PHP et distribuée sous licence GNU GPL.

Figma est un éditeur de graphiques vectoriels et un outil de prototypage. Il est principalement basé sur le web, avec des fonctionnalités hors ligne supplémentaires activées par des applications de bureau pour MacOs et Windows.

Adobe Illustrator est un logiciel de création graphique vectorielle. Il fait partie de la gamme Adobe,

Photoshop est un logiciel de retouche, de traitement et de dessin assisté par ordinateur, lancé en 1990 puis en 1992 pour les systèmes d’exploitation MacOS et Windows.

3-L’intelligence artificielle

ChatGPT, pour Chat Générative Pre-traine Transformer, est un prototype d'agent conversationnel utilisant l'intelligence artificielle, développé par OpenAI et spécialisé dans le dialogue. Il est basé sur les grands modèles de langage d'OpenAI GPT-3.5 et GPT-4.

BLACKBOX AI est un assistant de codage IA qui aide les développeurs en fournissant des suggestions de code en temps réel. Le moyen le plus rapide de copier du code à partir de vidéos et d'images et de rechercher des extraits de code depuis Jupyter ou Paperspace.

GitHub Copilot est une extension d'environnement de développement intégré basé sur l'intelligence artificielle et développé par GitHub et OpenAI pour aider les développeurs en complétant automatiquement le code

Microsoft Copilot est un chatbot développé par Microsoft. Basé sur un modèle de langage étendu, il a été lancé sous le nom de Bing Chat le 7 février 2023, en tant que fonctionnalité intégrée pour Microsoft Bing et Microsoft Edge, et constitue le principal remplacement par Microsoft de Cortana, abandonnée.
